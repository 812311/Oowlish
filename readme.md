# Run Application

Clone this project:
    HTTPS:
    `git clone https://github.com/812311/Oowlish.git`
OR
    SSH:
    `git clone git@github.com:812311/Oowlish.git`


To run the application you need to run 2 terminals:

First Terminal:
1 - `cd OowlishBack` 
2 - `yarn && yarn dev`
          OR
    `npm && npm dev`

Your server should be up and runnig on http://localhost.com:3333


Open the Second Terminal:
1 - `cd OowlishFront` 
2 - `yarn && yarn start`
          OR
    `npm && npm start`

Access [http://localhost.com:3000](http://localhost.com:3000)


All Set!
You should be able to use the aplication now!
